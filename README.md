# Cacao La Guajira

Proyecto de limpieza, validación, análisis y visualización de datos territoriales para aptitud de cacao en municipios de La Guajira.

## Objetivo
Desarrollar un proceso de limpieza, análisis exploratorio y construcción de dashboard en Power BI para caracterizar la aptitud territorial del cacao en el departamento de La Guajira.

## Contenido del repositorio
- `dashboard/`: archivo Power BI
- `data/raw/`: base de datos original
- `data/processed/`: base de datos limpia
- `notebooks/`: análisis en Python / Colab
- `reports/`: informe técnico
- `images/`: capturas del dashboard
